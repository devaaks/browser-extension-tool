YouTube Transcript Extension Debug

Test Instructions

  1. Load this extension in Chrome Developer Mode
  2. Navigate to a YouTube video page
  3. Open browser console (F12) and check for log messages
  4. Look for the extension badge on the toolbar
  5. Click the extension icon to open the popup

Debug Console Messages to Look For

  • YouTube Transcript Extension content script loaded
  • Initializing YouTube transcript extension on: [URL]
  • YouTube detection: {hostname, pathname, search}
  • Is YouTube video page: true/false
  • Extracted video ID: [VIDEO_ID]
  • Fetching transcript for video: [VIDEO_ID]

Troubleshooting

If no console messages appear:

  • Check that the extension is loaded and enabled
  • Verify the extension has permission for YouTube
  • Try refreshing the YouTube page
  • Check the Extensions page for any errors

If detection fails:

  • Check the URL format matches: youtube.com/watch?v=...
  • Wait a few seconds for the page to fully load
  • Try navigating to another video

Test Videos (Known to have transcripts)

  • Rick Astley - Never Gonna Give You Up
  • Luis Fonsi - Despacito
  • Any recent popular video should have auto-generated captions
